## 🚀 איך להשתמש?

### שלב 1: העתק קבצים
```bash
# העתק את המעבדים
cp excel_processor.py word_processor.py [PROJECT]/src/processors/

# העתק את מודולי התאימות
cp privacy_law_compliance.py ai_decision_engine.py [PROJECT]/src/
```

### שלב 2: התקן תלויות
```bash
pip install python-docx openpyxl xlrd
```

### שלב 3: השתמש בקוד
```python
# Excel
from excel_processor import ExcelProcessor
processor = ExcelProcessor()
result = processor.extract_text_from_excel("file.xlsx")

# Word
from word_processor import WordProcessor
processor = WordProcessor()
result = processor.extract_text_from_word("file.docx")

# תאימות
from privacy_law_compliance import PrivacyLawCompliance
checker = PrivacyLawCompliance()
compliance = checker.check_compliance(pii_results)

# החלטת AI
from ai_decision_engine import AIDecisionEngine
engine = AIDecisionEngine()
decision = engine.make_decision(pii_results, compliance_results)

# הצג תוצאות
print(f"החלטה: {decision.decision.value}")
print(f"ציון סיכון: {decision.risk_score}/100")
print(f"תקין: {compliance['compliant']}")
```
